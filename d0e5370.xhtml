5.123. simplified-ui-splash

5.123.1. Issues

5.123.1.1. Obsolete Migration Screen

1
The entire migration API is obsolete and will never be used. The splash screen still contains UI code for migrations.
2
Issue class: OBSOLETE. Severity: Minor.

5.123.1.2. Failure Mode Is Poor

1
If the application fails to start up due to a hard error in a service (which typically indicates a bug), the application shows an unhelpful error screen and suggests restarting. This will almost never fix the problem, and the user cannot see what went wrong and can only submit a rather opaque error report.
2
Issue class: DESIGN. Severity: Minor.

5.123.2. Module Link

1
The module can be found at simplified-ui-splash.
