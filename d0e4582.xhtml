5.97. simplified-presentableerror-api

5.97.1. Issues

5.97.1.1. Largely Obsolete

1
The contents of this module define an API for safely displaying error information onscreen in a way that indicates that the message text has been properly formatted and translated. However, the UI now never displays information in this way. Instead, an error details page is used that deliberately shows untranslated text because people tend to send screenshots of error messages to developers. This module could be removed.
2
Issue class: OBSOLETE. Severity: Minor.

5.97.2. Module Link

1
The module can be found at simplified-presentableerror-api.
