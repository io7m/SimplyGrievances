5.34.1. Issues

5.34.1.1. AutoValue AccountDescription

1
The AccountDescription class is still using an AutoValue generated type.
2
Issue class: AUTOVALUE. Severity: Minor.

5.34.1.2. AccountAuthenticationAdobeActivation Is Dead Code

1
The AccountAuthenticationAdobeActivation class is not referenced anywhere in the codebase and can be deleted.
2
Issue class: DEADCODE. Severity: Minor.

5.34.1.3. Unused AccountProvider idNumeric Field

1
The AccountProvider class exposes a (deprecated) idNumeric field that has not been used in the application since version 3.0.
2
Issue class: OBSOLETE. Severity: Minor.

5.34.1.4. AccountProviderAuthenticationDescription COPPA Check

1
Code exists in the AccountProviderAuthenticationDescription class that performs "age is over 13" checks for the COPPA age gate. Unfortunately, due to multiple instances of miscommunication, this check actually takes place in some new UI code and the code in the original class goes unused.
2
Issue class: OBSOLETE. Severity: Minor.
