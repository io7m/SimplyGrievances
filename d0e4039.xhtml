5.76.1. Issues

5.76.1.1. MainServices Class Is Too Large

1
The MainServices class that instantiates all of the application services is too large and should be broken up into separate classes. This issue relates to a lack of a general service registration mechanism.
2
Issue class: COMPLEX. Severity: Minor.

5.76.1.2. UI Thread Service Fundamentally Flawed

1
The MainUIThreadService class is fundamentally flawed and should be removed.
2
Issue class: UITHREAD. Severity: Critical.

5.76.1.3. MainBundledContentResolver

1
The MainBundledContentResolver class is no longer useful as noone is using bundled content in the application anymore.
2
Issue class: OBSOLETE. Severity: Minor.

5.76.1.4. Global Toolbar Handling

1
The application should probably not be using a global action bar. A per-tab Toolbar should probably be used instead.
2
Issue class: OBSOLETE. Severity: Minor.
