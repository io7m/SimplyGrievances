5.70. simplified-feeds-api

5.70.1. Issues

5.70.1.1. Leaky Abstraction

1
The module attempts to provide an abstraction over feeds, whether downloaded or generated locally. Unfortunately, the API leaks OPDS 1.2 data types into the feeds, making it difficult to move to OPDS 2.0 or other feed types in the future.
2
Issue class: DESIGN. Severity: Major.

5.70.1.2. Feed Groups Lose Ordering

1
The order of groups as they were declared in feeds is lost. This means that swimlanes in the application are shown in a non-deterministic order.
2
Issue class: DESIGN. Severity: Major.
3
Relevant issues: SMA-121.

5.70.2. Module Link

1
The module can be found at simplified-feeds-api.
