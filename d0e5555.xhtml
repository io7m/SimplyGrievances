5.128. simplified-viewer-epub-readium2

5.128.1. Issues

5.128.1.1. Page Turn Events

1
The viewer fails to publish page turned events to the analytics API. This is an oversight.
2
Issue class: COMPLEX. Severity: Minor.

5.128.1.2. Uses UIThread API

1
The module uses the fundamentally flawed UI thread API.
2
Issue class: UITHREAD. Severity: Critical.

5.128.1.3. Locked To Portrait Mode

1
The viewer is currently restricted to portrait mode due to possible lifecycle issues.
2
Issue class: LIFECYCLE. Severity: Minor.

5.128.1.4. Incorrect Use Of Fragment Factories

1
Due to an incorrect use of Fragment Factories (used to pass parameters to Fragments), the application will crash if the hosting activity is destroyed by the system when it is backgrounded. The correct approach should be to store parameters in a ViewModel and to require Fragments to be stateless and take no parameters.
2
Issue class: BUG. Severity: Major.
3
Relevant issues: SMA-203.

5.128.2. Module Link

1
The module can be found at simplified-analytics-api.
