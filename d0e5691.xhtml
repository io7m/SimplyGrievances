5.131. simplified-webview

5.131.1. Issues

5.131.1.1. Version-Specific Maintenance Burden

1
In order to support SAML authentication in an external WebView, custom code had to be written to extract cookies from the WebView's internal database. Multiple versions of this code are required to exist in order to handle all of the different version-specific database formats, and this represents a non-trivial maintenance burden.
2
Issue class: COMPLEX. Severity: Minor.

5.131.2. Module Link

1
The module can be found at simplified-webview.
