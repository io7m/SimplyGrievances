5.124. simplified-ui-theme

5.124.1. Issues

5.124.1.1. Action Bars Are Obsolete

1
The module contains references to theme variants with and without action bars. The application should be using a Toolbar and should not be using themes with action bars. Note that fixing this will likely affect external libraries such as the Audio Book modules.
2
Issue class: OBSOLETE. Severity: Minor.

5.124.2. Module Link

1
The module can be found at simplified-ui-theme.
