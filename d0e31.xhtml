This book represents a comprehensive examination of the issues with the Library Simplified Android client codebases. Specifically, problems are documented for the following codebases:

1.2. Documented Codebases

  • Simplified-Android-Core
  • audiobook-android
  • Simplified-R2-Android
3
The Issue Classes section documents the classes of issues seen in all of the codebases, and the Modules section describes specific instances of the issues in each module.
4
Additionally, this book attempts to identify all caches used in the applications, because caches are typically a source of many problems.
