5.111. simplified-ui-accounts

5.111.1. Issues

5.111.1.1. UI Needs Redesign

1
The UI for the accounts settings page as implemented in this module was designed to handle a single authentication method. The application now supports an arbitrary number of authentication methods, and the UI layout is creaking under the strain. Barring a complete redesign of the UI, the authentication methods should at least be wrapped in some form of carousel so the UI for that section can scale to any number of methods.
2
Issue class: DESIGN. Severity: Minor.

5.111.2. Module Link

1
The module can be found at simplified-ui-accounts.
