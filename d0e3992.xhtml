5.73. simplified-json-core

5.73.1. Issues

5.73.1.1. Lost Lexical Information

1
The module uses the Jackson Tree model to parse JSON. Unfortunately, the Tree model loses lexical information, so parse errors cannot indicate from where in the parsed document they originate.
2
Issue class: DESIGN. Severity: Major.

5.73.1.2. Fails On First Error

1
Code written using these methods tend to fail on the first error unless specifically designed not to do so. This increases the amount of labour required to fix all problems within a given document, as it's necessary to fix the first error, re-parse the file, fix the next error, re-parse the file, and so on.
2
Issue class: DESIGN. Severity: Minor.

5.73.2. Module Link

1
The module can be found at simplified-json-core.
