5.118. simplified-ui-navigation-tabs

5.118.1. Issues

5.118.1.1. Pandora BottomNavigator Is Buggy

1
The application uses the Pandora BottomNavigator library to manage tabs. This library has had numerous bugs in the two years that we've used it, and may be come unnecessary if the catalog navigation implementation is replaced.
2
Issue class: BUG. Severity: Minor.

5.118.1.2. Uses Fragment Backstack

1
Android's Fragment backstack is a fertile source of bugs, and has several implementation issues that will never be fixed. The backstack is used in our implementation of tabs, and ideally should not be used at all.
2
Issue class: DESIGN. Severity: Minor.

5.118.2. Module Link

1
The module can be found at simplified-ui-navigation-tabs.
