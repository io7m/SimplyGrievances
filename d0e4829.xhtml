5.104. simplified-reports

5.104.1. Issues

5.104.1.1. Email As Transport Method

1
Error reports are sent via email with log files as zipped attachments. This requires devices to have working email configurations and many do not. It also requires humans to manually forward error reports between developers. This should be replaced with a mechanism that uploads log files to a server over HTTP, with some kind of correlating session ID to identify specific crashes and problems.
2
Issue class: DESIGN. Severity: Major.

5.104.2. Module Link

1
The module can be found at simplified-reports.
