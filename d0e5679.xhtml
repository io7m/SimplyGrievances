5.129.1. Issues

5.129.1.1. Activity Listeners

1
The hosting Activity must communicate with Fragments using old-style callback functions. This should be replaced with communcation via a ViewModel (the PDF library predates ViewModels).
2
Issue class: STYLE. Severity: Minor.

5.129.1.2. Library Needs Update

1
The PDF library used is maintained by Minitex, and hasn't been updated for quite a while. It needs updating to use the latest versions of the various Android dependencies.
2
Issue class: OBSOLETE. Severity: Minor.

5.129.2. Module Link

1
The module can be found at simplified-viewer-pdf.
