5.58. simplified-books-database

5.58.1. Issues

5.58.1.1. Book Database Uses OPDS 1.2 JSON

1
The book database uses a slightly odd JSON serialization of OPDS 1.2 to store the state of each book. This is slightly awkward to maintain, and causes OPDS 1.2 semantics to leak into what should be a much more general information store.
2
Issue class: DESIGN. Severity: Minor.

5.58.1.2. Book Database Leaks Temporary Files

1
The book database appears to leak temporary files during some operations, causing storage use to increase very slowly over time.
2
Issue class: BUG. Severity: Minor.
3
Relevant issues: SMA-128.

5.58.2. Module Link

1
The module can be found at simplified-books-database.
