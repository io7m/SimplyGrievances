5.93. simplified-opds-core

5.93.1. Issues

5.93.1.1. Complicated Parser Code

1
The module contains rather tangled, complicated, and organically grown code to parse OPDS 1.2 streams. It's unclear if it correctly handles any of OPDS 1.2 in a compliant manner. It is also some of the oldest code in the application, and is very hard to understand.
2
Issue class: COMPLEX. Severity: Major.
3
Relevant issues: SMA-152.

5.93.1.2. Hand-Written Data Classes

1
The module contains many hand-written data classes. These should ideally be replaced with Kotlin data classes.
2
Issue class: OBSOLETE. Severity: Minor.

5.93.1.3. Availability Inference May Be Incorrect

1
The code may incorrectly infer availability information for OPDS feed entries. This may cause books to incorrectly appear to be available for loan, or to be unavailable.
2
Issue class: COMPLEX. Severity: Major.
3
Relevant issues: SMA-190.

5.93.2. Module Link

1
The module can be found at simplified-opds-core.
